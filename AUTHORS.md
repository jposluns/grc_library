## Attribution posture

The library is released under [CC0 1.0 Universal](LICENSE), the most permissive public-domain dedication available. CC0 imposes **no attribution requirement**. Adopters are legally free to fork, adapt, and use the library without crediting the original author.

That said, the author would appreciate citation where adopters find it useful. Suggested forms (any of these is welcome; none is required):

- A brief acknowledgement line in an adopter's internal governance documents: "Derived from the Governance, Risk, and Compliance Documentation Library (Posluns, J., CC0 1.0 Universal, https://github.com/jposluns/grc_library)".
- A reference in academic or industry publications using the citation metadata published in [`CITATION.cff`](CITATION.cff). GitHub renders a "Cite this repository" button on the repository sidebar that produces APA, BibTeX, and other formats.
- A link back to the source repository.

These are requests, not licence conditions. The library remains CC0; the author waives all copyright and related rights to the extent permitted by law.

## How to cite

For machine-readable citation metadata, see [`CITATION.cff`](CITATION.cff) at the repository root. GitHub renders this as a "Cite this repository" button when viewing the repository on the web.

Example citation (APA-style):

> Posluns, J. (2026). *Governance, Risk, and Compliance Documentation Library* [Dataset]. CC0 1.0 Universal. https://github.com/jposluns/grc_library

Example citation (BibTeX):

```bibtex
@misc{posluns_grc_library_2026,
  author       = {Posluns, Jeffrey},
  title        = {Governance, Risk, and Compliance Documentation Library},
  year         = {2026},
  publisher    = {GitHub},
  howpublished = {\url{https://github.com/jposluns/grc_library}},
  note         = {CC0 1.0 Universal}
}
```
