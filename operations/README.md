## Domain Coverage

The operations domain covers:

- **Production security standards** — baseline controls for production systems including hardening, segmentation, access control, logging, and configuration management.
- **Network security** — network architecture security, segmentation standards, firewall governance, east-west traffic controls, and DMZ configuration.
- **Cloud security configuration** — cloud service configuration baselines, shared responsibility boundary documentation, cloud-native control alignment.
- **Physical security of IT infrastructure** — data centre physical access controls, media handling, environmental controls.
- **Change and configuration management** — change advisory process, configuration control, baseline management, emergency change procedures.
- **Security monitoring and SIEM operations** — alert management, triage, escalation, tuning, and evidence preservation.
- **Threat intelligence** — threat feed integration, indicator management, and intelligence-driven monitoring.
- **Patch management** — patch classification, cadence, emergency patching, and exception governance.
- **Endpoint management** — device compliance, endpoint detection, mobile device management, and device lifecycle.
- **Certificate authority management** — PKI governance, certificate lifecycle, root CA controls, and certificate inventory.
- **Media handling and transport** — secure media classification, transfer controls, and destruction procedures.
